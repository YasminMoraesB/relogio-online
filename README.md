# 🕐 Relógio — Página dinâmica por período do dia.

Site que exibe a hora atual e adapta visual e imagens de acordo com o período do dia (manhã, tarde ou noite).

---

## ✨ Funcionalidades

- Exibe a hora atual com efeito de digitação animada (cursor piscando)
- Troca a imagem de acordo com o período do dia
- Muda o background e as cores da página dinamicamente
- Layout responsivo com footer fixo na base da página

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (Jersey 10, Poppins, Quicksand, entre outras)

---

## 🚀 Como rodar

1. Clone ou baixe o repositório
2. Certifique-se de que as imagens estão dentro da pasta `imagens/`
3. Abra o arquivo `index.html` diretamente no navegador

> Não é necessário servidor ou instalação de dependências.
